<p>It's a pretty simple little thing, but let's break it down a little bit. Here's this mixin itself:</p>


```css
  @function makelongshadow($color) {
  $val: 0px 0px $color;
  @for $i from 1 through 200 {
    $val: #{$val}, #{$i}px #{$i}px #{$color};
  }
  @return $val;
}

@mixin longshadow($color) {
  text-shadow: makelongshadow($color);
}
```

<p>Basically the <code>makelongshadow</code> function loops through and gives you a shadow value that consists of a bunch of stacked shadows: <code>1px 1px $color, 2px 2px $color...</code> etc. Then the <code>longshadow</code> mixin simply calls that function and applies the value to a text shadow. You'd use it like this:</p>


```css
  h1 {
  @include longshadow(#ccc);
}
```

<p>The resulting css would look something like this:</p>


```css
  h1 {
  text-shadow: 0 0 #CCCCCC,
    1px 1px #CCCCCC,
    2px 2px #CCCCCC,
    3px 3px #CCCCCC,
    4px 4px #CCCCCC,
    ...
    196px 196px #CCCCCC,
    197px 197px #CCCCCC,
    198px 198px #CCCCCC,
    199px 199px #CCCCCC,
    200px 200px #CCCCCC;
}
```
